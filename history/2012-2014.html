History, 2012–2014

Origin

Milestones

  1. 2012 Start / Project

    책읽는귀족 출범 · ‘마광수 다시 보기 프로젝트’

    출판을 시작점으로 삼아, 한 작가를 ‘재발견’하는 방식으로 해석·큐레이션·기획을 축적하는 브랜드 방향을 선언했다.

    • 『멘토를 읽다』
    • 『별것도 아닌 인생』
    • 『모든 것은 슬프게 간다』
    • 『청춘』
  2. 2013 Build

    편집·기획 시스템 정비

    원고 선별–편집–제작–홍보의 기본 루틴을 고정하며, 이후 확장 가능한 출판 구조를 만들기 시작했다.

    • 『청춘』
    • 『나의 이력서』
    • 『상상놀이』
    • 『육체의 민주화 선언』
    • 『즐거운 사라』
    • 『가자, 장미여관으로』(개정판)
    • 『마광수의 유쾌한 소설 읽기』
  3. 2014 Direction

    브랜드의 톤 앤 매너 확립

    ‘인문·교양’ 기반의 기획 방향과 장기 프로젝트를 축적해가는 운영 방식이 이 시기에 자리 잡았다.

    • 『생각』
    • 『아라베스크』
    • 『행복 철학』
    • 『스물 즈음』
    • 『마광수의 인문학 비틀기』
