History, 2012–2014

Origin

Milestones

  1. 2012.02.17 Start

    책읽는귀족 출범

    ‘출판을 시작점으로 삼아, 콘텐츠와 기획을 축적하는 브랜드’라는 방향이 이 시기에 정해졌다.

  2. 2012 Project

    ‘마광수 다시 보기 프로젝트’ 진행

    한 작가를 ‘재발견’하는 방식으로, 출판을 단순 제작이 아니라 해석·큐레이션·기획의 작업으로 선언했다.

  3. 2013 Build

    편집·기획 시스템 정비

    이후 확장 가능한 구조를 만들기 위해, 원고 선별–편집–제작–홍보의 기본 루틴을 ‘작게’ 고정했다.

  4. 2014 Direction

    브랜드의 톤 앤 매너 확립

    ‘인문·교양’ 기반의 기획 방향과, 장기 프로젝트를 축적해가는 운영 방식이 자리 잡기 시작했다.
